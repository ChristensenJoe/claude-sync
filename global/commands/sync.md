# Sync Claude Configuration

Push your current Claude Code configuration (memories, commands, settings, CLAUDE.md,
and project memories) to your cloud config repo.

## Instructions

1. Run the sync push script:

   ```bash
   $HOME/claude-sync/scripts/claude-sync push
   ```

2. Report the output to the user — what was pushed and whether it succeeded.

3. If the push fails (e.g., merge conflict, network error), explain the issue
   and suggest how to resolve it.
